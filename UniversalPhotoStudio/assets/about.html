Photo Corner is an open source photo viewer application, currently we support flickr, instagram and 500px, and later, we may support more photo sources. Since this is an open source project, you can download the source code and build the apk yourself, but we will appreciate if you can buy it from google play store.

With this application, you can:
  • Browse awesome photos from the photo sources;
  • See your own flickr and instagram photos;
  • Share photos to other applications installed on your phone;
  • Comment or like photos of others;
  • Set wallpaper;
  • ...

When developing this application, we used several open source projects, we'd like to express our appreciations to the developers:
  • Sliding Menu
  • jInstagram
  • ArcMenu
  • Android-ViewPagerIndicator
  • Universal Image Loader

If you want to build the application from our source code, you need to know the following:
  • Get the lib projects mentioned in our GIT web page;
  • We modified jInstagram a little bit to support pagination and provide 'count' paramter;
  • The lib project to access 500px photos was developed by our own, but it's not ready to open source;

Thank you.
